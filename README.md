# Smart-Task-Manager


# Install Backend Dependencies...

npm i express mongoose cors dotenv { Backend Dependencies }

- express - web framework for nde.js (used to build APIs and handle routes)
- mongoose - ODM ( objet Data Modeling) library to interact with MongoDB
- cors -  Middleware to enable Cross-Origin Resources Sharing (frontend <-> backend communication)
- dotenv - Loads environment variables from a .env file (e.g., DB URL, PORT, secrets)



npm i nodemon --save-dev  { Dev Dependency }

- nodemon - Automatically restarts the server when file changes are detected (for faster development)
